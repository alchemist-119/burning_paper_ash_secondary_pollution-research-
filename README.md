# 🚨 Why This Research Matters

![Why This Research Matters](01_why_this_research_matters.png)

## 研究迫切性

每逢重要節慶，燒金紙活動會產生大量飛灰。雖然煙霧會逐漸消散，但殘留灰燼可能沉降於土壤，並在降雨作用下形成二次污染，進一步影響土壤環境與植物生長。

目前多數研究聚焦於燃燒過程中的空氣污染，對於飛灰沉降、雨水沖刷及植物生長的完整影響仍缺乏系統性的探討。

本研究希望建立完整且可重複的研究流程，提供環境教育、污染防治及後續研究的重要參考。

# 📖 Project Overview

![Project Overview](02_project_overview.png)

## 專案介紹

本研究將以三個階段完成：

🌱 第一季：分析飛灰是否改變土壤理化性質。

🌧 第二季：模擬雨水沖刷造成二次污染。

🌿 第三季：分析污染土壤對植物生長的影響。

最終建立完整的二次污染傳遞模式。

# 🌱 Research Hypothesis & Expected Outcomes

![Research Hypothesis & Expected Outcomes](05_expected_outcomes.png)

## 研究假說

本研究提出以下假說：

- 飛灰可能改變土壤理化性質。
- 雨水可能造成污染物向下滲流。
- 土壤改變可能影響植物發芽與生長。
- 最終建立完整二次污染傳遞模式。
