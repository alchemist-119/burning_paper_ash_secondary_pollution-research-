# 👋 我想邀請大家一起完成這個 Project

每逢重要節慶，

我們都會看見燒金紙所產生的煙霧。

可是，

我開始思考一件事情。

**煙霧散去之後呢？**

那些灰燼最後去了哪裡？

如果灰燼落到土壤，

再遇到下雨，

會不會形成另一種我們看不見的污染？

目前，大部分研究都聚焦於燃燒時的空氣污染，

但「灰燼沉降 → 雨水沖刷 → 土壤改變 → 植物生長」

這條完整的污染路徑，

卻很少有人真正去驗證。

所以，

我想把這個問題，

變成我們一起完成的 Project。
