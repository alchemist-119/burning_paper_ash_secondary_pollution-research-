# 🌱 我們想解開一個生活中的謎題

如果一粒灰燼，

離開香爐之後，

它會去哪裡？

會留在土壤？

會跟著雨水移動？

會影響植物嗎？

我們不知道。

所以，

想和大家一起找答案。

# ❓ 為什麼值得研究？

生活中的每一天，

環境都在默默改變。

如果能了解灰燼在環境中的旅程，

就能更認識：

🌱 土壤

💧 水

🌿 植物

也提醒我們，

一起關心生活環境，

以及與生活息息相關的食物來源。
