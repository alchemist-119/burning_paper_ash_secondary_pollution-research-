# 🔍 Project Overview

燒金紙，

是生活中熟悉的一部分。

我們尊重文化，

也尊重信仰。

我們想研究的，

不是誰對誰錯。

而是：

**灰燼最後去了哪裡？**

它會停留在土壤嗎？

會跟著雨水流動嗎？

植物會留下它的痕跡嗎？

不知道，

就一起尋找答案。

# 🧪 Research with Accessible Tools

好的研究，

不一定需要昂貴的設備。

我們希望建立一套：

- 可以重複
- 可以分享
- 可以讓更多孩子參與

因為，

真正重要的是研究方法，

而不是設備價格。
