# 🌱 一切，從生活開始

今年，

草原鹿想一起完成一場公民科學探險。

我們沒有從課本開始，

而是從生活開始。

每天，

我們都會看見許多習以為常的事情。

直到有一天，

我們開始問：

> **「為什麼？」**

科學，

就從這一刻開始。

# 🔍 我們選擇了一個生活中的問題

燒金紙，

是生活中熟悉的一部分。

我們尊重每一種文化，

也尊重不同的信仰。

我們想了解的，

不是誰對誰錯。

而是：

**灰燼最後去了哪裡？**

它會停留在土壤嗎？

會跟著雨水流動嗎？

植物會留下它的痕跡嗎？

不知道，

就一起尋找答案。

---

## 🌏 為什麼選擇這個題目？

真正的公民科學，

常常不是從遙遠的地方開始。

而是從生活中，

那些習以為常的事情開始。

# 🔬 我們如何尋找答案？

我們相信，

答案不是猜出來的。

而是一步一步，

慢慢找出來的。

🌱 觀察生活

# 💡 我們真正想學會的是……

比起完成一份研究，

我們更希望學會：

✔ 提出好問題

✔ 尊重不同觀點

✔ 與夥伴合作

✔ 勇敢分享發現

會一直陪伴我們。

# 🏫 把發現分享出去

如果研究順利完成，

我們希望把成果分享給更多人。

不是告訴大家：

**「你應該怎麼做。」**

**「這是我們一步一步找到的答案。」**

因為分享，

才能產生更多新的想法。
