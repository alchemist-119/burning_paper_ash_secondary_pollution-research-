# 🌱 Why This Research Matters

每天，

# 🔍 Project Overview

燒金紙，

是生活中熟悉的文化傳統。

我們尊重不同信仰，

也尊重不同文化。

因此，

我們並不是想證明誰對誰錯。

我們只是想了解：

**一粒灰燼，離開香爐之後，會經歷什麼旅程？**

它是否可能：

🌱 留在土壤？

🌧️ 跟著雨水流動？

🌿 影響植物生長？

當我們一步一步理解環境中的變化，

也希望思考一個更大的問題：

**環境中的二次污染，是否可能間接影響我們每天生活的環境，甚至未來的食物來源？**

這不是結論，

而是值得一起探索的問題。

# 🌱 Expected Outcomes

一年後，

我們希望留下的不只是研究結果。

更希望留下：

✔ 一份可信的研究紀錄

✔ 一套可以分享的研究方法

✔ 一群願意觀察生活的孩子

✔ 一個讓更多人開始關心環境的故事

如果，

這份研究，

能讓更多人開始思考：

環境中的變化，

是否也可能一路影響植物、

影響生態，

甚至值得我們持續關心生活環境與食物來源。

那麼，

這場公民科學探險，

就已經成功了。

# ❤️ From One Question to Many Discoveries

一年前，

我們只有一個問題。

一年後，

我們得到的，

也許不只是答案。

更重要的是，

我們學會：

🌱 用好奇開始研究

🔍 用證據回答問題

🤝 與夥伴一起合作

📖 與更多人分享發現

因為，

真正的公民科學，

不是告訴大家答案。

而是邀請更多人，

一起關心我們共同生活的環境。

也一起思考：

今天環境中的每一個小小改變，

是否都可能影響明天的世界。
